# Finance App

### Get Started

1. Create Database
2. Create connection with Database
3. To run sql
```
console.sql
```
4. To run in terminal
```
npm i
```
5. Change obj in file app.js
```
const pool = new Pool({
    user: "",
    password: "",
    host: "",
    port: "",
    database: ""
});
```
6. To run the program
```
node app.js
```
7. Go to
```
localhost:8080
```
